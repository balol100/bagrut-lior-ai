# bagrut.lior-ai.com

תרגול בגרויות חינם — single-file HTML app with 165+ Bagrut practice questions across 6 subjects, with detailed Hebrew step-by-step solutions.

**Subjects:** מתמטיקה (3/4/5 יח'), אנגלית (3/4/5 יח'), היסטוריה, אזרחות, תנ״ך, מדעי המחשב (3/5 יח').

**Stack:** Single `index.html` — no build, no dependencies. Service worker for offline. RTL Hebrew throughout.

**Deploy:** Drop `index.html` on Netlify (or any static host).
